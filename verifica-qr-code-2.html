Carica una tabella Excel per alimentare il programma, oppure crea una scheda a mano.

Inquadra il QR code. Se il codice è già in archivio si apre la scheda, altrimenti puoi crearne una nuova con quel codice.
Premi “Avvia fotocamera”.
Importa
Carica un file .xlsx / .xls / .csv. La colonna del codice viene riconosciuta automaticamente tra: Matr., SN, Serial Number, Seriale, Numero di Serie, QR. Le altre colonne (ID, Descrizione, Ubicazione…) vengono mantenute e restano modificabili.

Esporta
L'esportazione genera un Excel con numero progressivo nel nome (es. archivio_v03.xlsx) per distinguere le versioni. Le foto si esportano a parte in un file ZIP, con nome basato sul codice dell'oggetto.
